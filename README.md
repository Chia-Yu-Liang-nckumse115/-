### 介紹

這個小專案從一個很單純的念頭開始：「目前市售的牛排溫度計可以告訴我牛排中心幾度，但我又不是專業廚師我怎麼知道幾度之下長怎樣？然後根據大家的習慣也有不同熟度，我就想看橫切面」  
所以基本上我設計一個元件-《智慧牛排熟度偵測探針》
目標是讓你不用切開牛排，也能看到裡面的熟度，讓新手都可以間出自己理想、客製化的牛排

運作方式主要是探針上分佈五個偵測點，能量測牛排從表面到中心的溫度梯度，並透過溫度–熟度模型重建牛排「內部剖面的顏色分佈」。使用者無需切開肉品，即可直觀判斷熟度（如三分、五分、七分熟），並在達到預設門檻時收到提醒。

概念:五個或更多的偵測點形成一維溫度剖面（1D thermal profile）程式即時進行 溫度→顏色 映射，重建「虛擬切面」
1. 物理探針：螺旋 15 點量測
   探針插入牛排後，會得到：T0, T1, T2, ..., T14
   這些點沿著螺旋分佈，因此代表：從外殼（表皮）→ 中心 → 更深處。有深度，也有橫向位移(會更準)
2. 從 15 點 →推測連續 1D 溫度函數(因為牛排加熱是連續的所以不需要完全密集的溫度數值，這樣也省錢)
3. 用 1D 溫度曲線 → 推導 2D 橫截面溫度場
4. 將重建的 T(x,y) → 光譜温度圖
   Normalize 溫度到 0~1
   套 colormap（這邊用jet）
5. 再把光譜圖 → 還原成虛擬牛排影像

元件的設計如下:
做一支可插入牛排的「螺旋溫度探針」，沿螺旋路徑放 15 個溫度感測點，
用多工電路讀出 1D 溫度剖面，再跟影像的 pseudo-temperature map 校正，
最後重建還原出比較真實的橫截面溫度場。
